Добро пожаловать в официальную документацию архитектуры процессора Amalthea. Здесь вы можете узнать больше о нашей архитектуре и даже приобрести новые навыки.
## Быстрый старт:
1. Приобретите или получите по почте процессор KiG TN P0/1, либо купите FPGA Tang Nano 9k (что будет гораздо больше мороки), а затем подключите плату/процессор к компьютеру через USB‑C.

2. Установите Gowin EDA с официального сайта (www.gowinsemi.com), включая все драйверы, предлагаемые загрузчиком (обязательно). После установки на рабочем столе появится ярлык программы — дважды щёлкните по нему.

Теперь необходимо скачать официальный архив архитектуры процессора. Он доступен в нашем Telegram‑канале по ссылке: (https://github.com/Kig-Organization/Amalthea-Archive.git).

Откройте Gowin EDA и в окне Quick Start дважды щёлкните на Open Project, затем выберите папку архива.
![example](docs/photo_5431611626634284618_w.jpg)

Теперь вы находитесь в окне проекта. В верхней панели выберите Run Synthesis, затем Run Place & Route. После этих действий в той же панели нажмите Programmer (рядом с Run Synthesis).
![example](docs/photo_5431611626634284432_w.jpg)

(Gowin EDA автоматически определит ваше устройство). В поле Frequency выберите 15 MHz и нажмите Save. После этих действий в верхней панели окна Programmer выберите Program/Configure, и через несколько секунд битстрим архитектуры будет загружен в вашу плату/процессор!
Если всё прошло успешно, на вашей плате должны загореться третий и нижний светодиоды:
![example](docs/photo_5431611626634284602_y.jpg)

(В некоторых случаях загорается только третий диод — это абсолютно нормально, если он не мигает и при прошивке не возникло ошибок. Просто игнорируйте это.)
