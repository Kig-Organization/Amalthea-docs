# Full list of operations in Amalthea

## Basic Arithmetic Operations

ADD — addition of the accumulator with an operand

SUB — subtraction

MUL — multiplication (if resources allow)

DIV — division (often implemented sequentially to save logic)

## Logical Operations

AND — bitwise AND

OR — bitwise OR

XOR — exclusive OR

NOT — bitwise negation

## Data Transfer

LOAD — load operand into the accumulator

STORE — write accumulator to memory/register

MOV — copy between registers

## Control

JMP — unconditional jump

JZ / JNZ — conditional jumps (based on zero flag)

HLT — halt
